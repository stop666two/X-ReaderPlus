完全脱机的现代化多格式电子书阅读器，首个正式版本。

## 功能

### 书架
- 网格 / 列表双视图，封面尺寸三档可调
- 拖拽批量导入，复制导入 / 文件夹引用两种模式
- 多书库管理，全部书库聚合视图
- 书名作者搜索、标签筛选、7 种排序方式
- 批量选择（全选 / 反选 / 删除），删除进回收站
- 书籍详情：评分、短评、标签编辑

### 阅读器
- 16 种格式支持：EPUB / TXT / Markdown / HTML / MOBI / AZW3 / FB2 / DJVU / DOCX / RTF / ODT / PDF / CBR / CBZ / CBT / CB7
- 分章节阅读，智能章节检测（中文 / 英文 / 序言 / 番外等）
- 智能作者识别（扫描前 40 行 + 后 15 行）
- 自动标签识别（#标签 / 标签：等格式，前 100 行）
- 章节导航：按钮 / 键盘方向键 / 屏幕边缘点击 / 底部滑块
- 阅读位置持久化——关闭重开回到精确位置，跨会话记忆
- 已读章节打勾，目录面板实时标记
- 专注模式——仅高亮当前段落，其余渐隐
- 排版微调：字号（12-32）、行距、段距、缩进、字重、字体、边距、页宽
- 3 种主题：明亮 / 暗黑 / 护眼，主题颜色可深度自定义
- Ctrl+滚轮缩放，工具栏 4 秒自动隐藏
- 全文搜索（关键词高亮 + 结果导航）
- 自动滚屏（可调速）
- 虚拟滚动——万行长章节流畅不卡
- 阅读统计浮窗：阅读速度、预计剩余、全书进度

### 标注与笔记
- 划词高亮（6 色：黄绿蓝红橙紫），颜色即时生效
- 划词笔记——给选中的文字写批注
- 复制 / 查词典（在线 + 内置 250+ 词中英互译）
- 全局笔记管理——按书 / 关键词筛选
- 笔记跳转定位——红色背景居中高亮原文位置
- 笔记回收站——软删除可恢复
- 批量选择、编辑、删除

### 词典
- 书库全文搜索——标题 / 标签 / 正文 / 笔记可选范围
- 在线英文词典（dictionaryapi.dev）
- 内置 250+ 词中英互译词典
- 词性彩色徽章，最近查询记录

### 安全
- **完全脱机**——数据 100% 存储本地 IndexedDB
- **PIN 码保护**——SHA-256 + PBKDF2（60 万次迭代）+ AES-256-GCM 加密
- 两级锁定策略——可配置错误次数和锁定时间
- 安全问题重置 PIN
- HTML 净化（DOMPurify）+ 严格 CSP

### 桌面特性
- 无边框自定义窗口 + 系统托盘
- 老板键（Ctrl+B）一键隐藏
- 命令面板（Ctrl+K）全局搜索
- 开机自启 + 启动最小化到托盘
- WebDAV 云备份——自动备份（每天/每周/每月）
- 数据导出 / 导入（加密备份）

### 其他
- 阅读统计（藏书量、总字数、格式分布、阅读时长）
- 阅读历史 + 已删除标记
- 启动加载进度条
- Web Worker 多线程解析——导入不卡 UI
- 内容去重——SHA-256 哈希检测重复书籍
- 10 种编码自动检测
- 批量分块操作——删除数千本不卡
