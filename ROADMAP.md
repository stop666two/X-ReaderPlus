# Roadmap — X-ReaderPlus

本文档记录 X-ReaderPlus 的功能规划与开发路线。

---

## v0.4.1 ✅ 已发布（当前）

**全量代码审计修复** — 修复 120+ 项问题

- [x] 安全：XSS 加固(DOMPurify)、CSP/CORS/安全头、AES 密钥即时派生
- [x] 安全：ReadFile/FileSizes 路径校验、POST 10MB 体限制、CBT 死循环修复
- [x] 功能：专注模式重写(RAF动画循环→滚动居中)、标注跳转、书库卡片重写
- [x] 功能：清理失效数据按钮、导出进度条、笔记来源缓存
- [x] 功能：系统设置(启动延迟/托盘行为/桌面通知)
- [x] 性能：LRU缓存、导入并发3、搜索索引同步写入
- [x] 构建：wails build 全平台、SVG图标、CI/CD更新
- [x] 数据：便携式 dataDir（exe同目录）

---

## v0.5.0 🚧 计划中

### 安全与稳定性

- [ ] **字典 API 隐私保护** — 词典请求改走后端代理 + 本地离线词典缓存（审计 C-4）
- [ ] **浏览器开发模式安全** — 敏感数据不落 localStorage（审计 H-4）
- [ ] **WebDAV 密钥加密存储** — 使用系统 Keychain/DPAPI 包装密钥（审计 H-5）
- [ ] **批量操作事务** — deleteBooks/回收站跨库事务保护（审计 H-6/H-15）
- [ ] **handleClear 跨库原子性** — Content/Settings 也纳入事务（审计 H-1/H-3）

### 功能增强

- [ ] **PDF 文本选择与缩放优化**
- [ ] **EPUB3 完整支持**（MathML, SVG 多媒体）
- [ ] **词典增强**（本地离线词典 + 历史缓存）
- [ ] **Go 后端格式解析器**（减少前端依赖）
- [ ] **漫画模式双页浏览**
- [ ] **语音朗读（TTS）**

### 代码质量

- [ ] **ESLint 全面修复** — 修复警告，移除 `--quiet`，添加浏览器全局变量
- [ ] **空 catch 清理** — 关键路径添加错误处理
- [ ] **搜索索引重构** — 增量更新替代全量序列化（M-18）
- [ ] **i18n 完善** — 确保 en-US/zh-CN 所有 key 同步
- [ ] **FB2/DJVU/CHM 解析器优化** — 正则回溯、字符串拼接、字节扫描
