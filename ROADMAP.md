# X-ReaderPlus 发展规划

> 核心原则不变：完全脱机、数据本地、简洁实用、隐私至上。以下所有计划均不依赖网络。

---

## v0.2.0 — 格式 & 性能（短期）

- [x] **阅读时间统计修复**：增量保存替代重复计数
- [x] **漫画格式声明修正**：CBR/CBT/CB7 显示格式说明
- [x] **DJVU 文本解压修复**：DecompressionStream 正确解压
- [x] **文件路径安全验证增强**：Windows 系统目录拦截
- [x] **文件读取 Buffer 截断修复**：精确传输数据
- [x] **EPUB 内嵌图片渲染**：正文图片提取为 data URI 内嵌
- [x] **标注偏移量精确计算**：Range API 替代 indexOf 搜索
- [x] **阅读计时器窗口隐藏暂停**：visibilitychange 事件暂停
- [x] **EPUB 封面路径解析**：绝对路径相对于 rootDir 解析
- [x] **导出备份含加密密钥**：换设备恢复可正确解密
- [x] **主题加载无闪烁**：移除 immediate watch
- [x] **书架页面免重复加载**：检查数据是否已就绪
- [x] **背景图片 URL 验证**：仅允许 http/https/data URI
- [x] **图标路径修正**：icon.png → icon.svg
- [x] **日志统一**：console.error → logger
- [x] **EPUB 内嵌图片渲染（增强）**：SVG支持 + ../路径规范化 + guide封面回退
- [x] **漫画模式**：CBZ/CBT 图片翻页，CBR/CB7 提示转换
- [x] **PDF Canvas 渲染**：逐页渲染为PNG，IntersectionObserver按需加载
- [x] **MOBI/AZW3 非 ASCII 编码**：UTF-8/GB18030 解码替代逐字节 ASCII
- [x] **长章节懒渲染**：IntersectionObserver 分段加载，10万字以上不卡
- [x] **全文索引**：章节内容预建倒排索引，大书库搜索秒级响应
- [x] **书架性能优化**：封面压缩缩略图 + 分页50本 + content-visibility
- [x] **快捷键可视化面板**：kbd 标签展示 + 冲突检测
- [x] **独立元数据数据库**：meta 表分离存储，列表加载提速
- [x] **全视图分页**：书架/笔记/标签/历史/回收站/统计均可翻页
- [x] **跨页批量选择**：第1页+第2页选择合并操作，反选仅限当页
- [x] **删除当页数据**：历史/回收站/统计均支持仅删当页
- [x] **分页设置自定义**：各视图每页条数独立配置（10-100）
- [x] **更多编码支持**：ISO-2022-JP、EUC-TW，覆盖更多老旧 TXT 文件（提前完成）
- [x] **标注导出**：标注导出为 Markdown/CSV（提前完成）
- [x] **数据库迁移 SQLite**：IndexedDB→SQLite，WAL模式，IPC通信，24文件重写
- [x] **书库管理页面**：独立管理页，支持增删改+隐私锁
- [x] **隐私锁系统**：书库/书本独立密码，8位强密码，PBKDF2+AES-256-GCM
- [x] **单本书导出**：右键菜单+详情弹窗导出完整JSON
- [x] **导入选书库**：复制导入可选目标书库+新建书库
- [x] **图标系统**：任务栏+托盘32x32蓝色书本PNG，窗口SVG渐变书本
- [x] **v0.2.0 审计修复**：9 Critical + 12 High 修复，IPC safeHandler 全覆盖，导航阻塞修复，书签/标注数据修复，封面压缩启用，空 catch 日志化，electronAPI 空安全，全局异常处理，导航安全防护（2026-06-25）
- [x] **阅读位置保持**：onBeforeRouteLeave await保存，切换侧边栏不丢失

## v0.3.0 — 阅读体验（中期）

- [ ] **多标签页**：同时打开多本书，标签切换
- [ ] **Calibre 书库导入**：直接读取 Calibre 的 metadata.db，批量导入
- [ ] **自定义阅读 CSS**：用户可写 CSS 完全控制排版（已有基础，需完善）
- [ ] **TTS 朗读**：调用系统自带语音引擎，完全离线，可调速
- [ ] **WebDAV 增量同步**：仅同步变化的标注和进度，而非全量备份

---

## v0.4.0 — 扩展 & 生态（长期）

- [ ] **插件系统**：允许第三方开发本地插件（格式解析器、主题、工具）
- [ ] **EPUB 3.0 完整支持**：MathML、Media Overlay
- [ ] **阅读数据可视化**：阅读热力图、习惯统计
- [ ] **移动端**：iOS/Android 版本，与桌面端共享数据格式
- [ ] **多语言国际化**：英文/日文/韩文界面

---

## 不打算做的

- ❌ 需要联网的功能（RSS、在线补全元数据、OPDS、云同步）
- ❌ DRM 破解
- ❌ 在线书城、付费内容
- ❌ 社交功能、排行榜、评论
- ❌ AI 功能（即使本地也违背"简洁"原则）
- ❌ 订阅制、付费墙
